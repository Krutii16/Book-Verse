================================================================================
                        BOOKVERSE - START HERE
================================================================================

Welcome to BookVerse! A complete e-commerce book platform built with Angular
and Express.js.

================================================================================
                            QUICK START (5 min)
================================================================================

STEP 1: Install Dependencies
  $ npm run install-all

STEP 2: Seed Database (creates 50+ sample books)
  $ npm run seed

STEP 3: Start Application
  $ npm run dev

STEP 4: Open Browser
  Go to: http://localhost:4200

That's it! The app is now running.

================================================================================
                          TROUBLESHOOTING
================================================================================

MongoDB Connection Error:
  → Make sure MongoDB is running (mongod)
  → Check MONGODB_URI in server/.env

Port Already in Use:
  → Change PORT in server/.env
  → Or kill process: kill -9 <PID>

Frontend Not Loading:
  → Check browser console (F12)
  → Verify backend is running
  → Check API URL in client/src/environments/environment.ts

Can't Login:
  → Verify email and password
  → Try admin@bookverse.com / admin123
  → Run npm run seed to reset

================================================================================
                          API ENDPOINTS
================================================================================

Books:
  GET /api/books              - Get all books
  GET /api/books/:id          - Get book details
  GET /api/books/search       - Search books
  GET /api/books/filter       - Filter books

Cart:
  POST /api/cart/add          - Add to cart
  GET /api/cart               - Get cart items
  DELETE /api/cart/:itemId    - Remove from cart

Orders:
  POST /api/orders            - Create order
  GET /api/orders             - Get user orders

Authentication:
  POST /api/auth/register     - Register user
  POST /api/auth/login        - Login user
  GET /api/auth/me            - Get current user

(See API_IMPLEMENTATION_GUIDE.md for full list)
