## 🎯 Quick Start Template

If you just want to START IMMEDIATELY:

```bash
# 1. Read START_HERE.md (10 min)

# 2. Run these commands:
cd BookVerse

# Terminal 1: Database
mongosh
use bookverse
# (paste setup commands from MONGODB_SETUP.md)

# Terminal 2: Backend
cd backend
npm install
npm run dev

# Terminal 3: Frontend
cd bookverse-frontend
npm install
ng serve

# 4. Open http://localhost:4200 in browser

# 5. You're running! Now read other guides as you code
```
